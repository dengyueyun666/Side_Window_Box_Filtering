# Side_Window_Box_Filtering
This is a C++ implementation of Side Window Box Filtering based on official Matlab code from [here](https://github.com/YuanhaoGong/SideWindowFilter).

## Requirements
* OpenCV3
* Armadillo-9.800.3+
